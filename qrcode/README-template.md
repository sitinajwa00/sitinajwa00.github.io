### What I learned

From the sample design, I have to ensure that the qr code is positioned at the center of the page. The css for this style is:

```css
.qr {
    position: absolute;
    top: 50%;
    left: 50%;
    transform: translate(-50%, -50%);
    padding: 10px;
    width: 350px;
}
```
